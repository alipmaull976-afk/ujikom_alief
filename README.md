🧾 Aplikasi Kasir Sederhana (Ujikom)

📖 Deskripsi
Aplikasi kasir berbasis PHP yang digunakan untuk mengelola data barang dan melakukan transaksi penjualan.
Project ini dibuat untuk keperluan Ujian Kompetensi (Ujikom).

⚙️ Fitur Utama
- 🔐 Login Admin & Kasir
- 📦 Kelola Data Barang (Tambah, Edit, Hapus)
- 🛒 Transaksi Penjualan
- 🧾 Cetak Nota
- 📊 Riwayat Transaksi

🛠️ Teknologi yang Digunakan
- PHP Native
- MySQL
- HTML, CSS, JavaScript

📂 Struktur File Penting
- "config.php" → Koneksi database
- "login.php" → Halaman login
- "admin_dashboard.php" → Dashboard admin
- "kasir_dashboard.php" → Dashboard kasir
- "daftar_barang.php" → Menampilkan data barang
- "tambah_barang.php" → Menambah barang
- "edit_barang.php" → Edit barang
- "hapus_barang.php" → Hapus barang
- "kasir_transaksi.php" → Halaman transaksi
- "cetak_nota.php" → Cetak struk
- "riwayat_transaksi.php" → Riwayat transaksi
- "penjualan.sql" → Database

🚀 Cara Menjalankan Project
1. Install XAMPP / Laragon
2. Ekstrak project ke folder "htdocs"
3. Buka phpMyAdmin
4. Import file "penjualan.sql"
5. Jalankan di browser:
   http://localhost/ujikom_alief
